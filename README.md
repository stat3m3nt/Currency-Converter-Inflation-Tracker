# 💱 Currency Converter & Inflation Tracker App

## 📱 Overview

This is a mobile application built using Expo (React Native) that allows users to convert currencies in real time and analyze inflation trends using global economic data.

The app integrates external APIs to provide accurate financial data and help users understand changes in purchasing power over time.

---

## 🚀 Features

* 💱 Real-time currency conversion
* 📊 Inflation tracking using historical data
* 🌍 Support for multiple currencies
* 📈 Compare value of money across different years
* 📱 Mobile-friendly UI built with Expo

---

## 🛠️ Tech Stack

* React Native (Expo)
* JavaScript
* REST APIs

---

## 🔗 APIs Used

* ExchangeRate.host API
  Used to fetch real-time and historical currency exchange rates.

* World Bank API
  Used to retrieve inflation data and economic indicators.

---

## ⚙️ How It Works

* The app sends API requests to retrieve live exchange rates.
* Inflation data is fetched and processed to calculate changes in purchasing power.
* Results are displayed dynamically in the user interface.

---

## ▶️ Run Locally

```bash
npm install
npx expo start
```
